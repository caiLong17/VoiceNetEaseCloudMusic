# VoiceNetEaseCloudMusic

---

网易云音乐语音版-基于mpv播放器与讯飞语音识别。

## 平台与依赖包  

平台：Windows  
依赖包：可通过``pip install -r requirements.txt``进行安装。

## 运行方法  

``python client.py``

## 命令行支持命令 
| command     | action     |
| :----:      | :----:     |  
| play        | 开始播放   |   
| pause       | 暂停播放   |   
| resume      | 恢复播放   |
| next        | 下一首     |
